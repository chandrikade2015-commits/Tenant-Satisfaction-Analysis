# Tenant Satisfaction Analysis

## Overview
This project analyzes tenant satisfaction across multiple buildings to uncover key factors that influence overall satisfaction.  
The analysis includes factors such as maintenance response time, support programs, and neighbourhood crime rate.

## Objective
Identify actionable insights to improve tenant satisfaction and operational efficiency.

## Tools Used
- Power BI
- Excel
- DAX / Power Query
- Data Cleaning & Transformation

## Key Insights
- Faster maintenance response times strongly correlate with higher satisfaction.
- Buildings with more support programs and lower neighbourhood crime rates show better tenant feedback.
